Details
=======
It is possible to use post-processing anti-aliasing, namely SMAA (Enhanced Subpixel Morphological Anti-Aliasing).
Version used: 2.7

Settings
========
In Arma3Alpha.cfg there is a config entry: PPAA=<number>;

Accepted values are:
 0 - Disabled
 4 - SMAA_PRESET_LOW_
 5 - SMAA_PRESET_MEDIUM
 6 - SMAA_PRESET_HIGH
 7 - SMAA_PRESET_ULTRA Complexity of quality settings are based on default SMAA settings (in revision 2.7)

NOTE: SMAA can be enabled even if Post-Processing was disabled in video options!

Sources
=======
Read more: http://www.iryoku.com/smaa/#downloads
